# Karousel

## Getting Started

Install w/ NPM:
```
npm i karousel
```

Install w/ Yarn:
```
yarn add karousel
```

## Basic Usage

```
import { Karousel } from 'karousel';

export const MyComponent = () => {
  return (
    <Karousel>
      <span>Item 1</span>
      <span>Item 2</span>
      <span>Item 3</span>
      <span>Item 4</span>
      <span>Item 5</span>
    </Karousel>
  );
}
```
